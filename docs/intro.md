# PACK

PACK (Platform Application Capabilities Kit) is a framework for building rich, collaborative applications within the Foundry developer platform. It is designed to be used alongside OSDKs and provides a storage mechanism for application state, application activity, per-user preferences, and multiplayer presence.

PACK is the evolution of battle-tested real-time frameworks used internally at Palantir to build collaborative applications. It's designed for multiplayer apps where everyone needs the same view of data in real time.
